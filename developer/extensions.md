# Create extensions

Extensions add supported metadata or behavior without copying the base app. Declare the target app dependency, use unique names, and keep the extension removable.

Create the extension with the CLI, add only the changed fields/forms/menu items or logic, then test with and without other extensions enabled. Do not overwrite framework files or depend on registration order that is not declared.
